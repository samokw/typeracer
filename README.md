## Getting Started

### Dependencies
Youll need to have:
Python 3.10, and NodeJs installed

#### Python Libaries

### Backend
* To start the flask server
```
python api.py
```
* Frontend
```
cd frontend
```
```
npm install
```
```
npm run dev
```
